Student Expected Outcomes:

Upon completion of this course, the student will be able to:

  • Understand and become familiar with basic UNIX commands and the ability to combine them to accomplish system tasks.
  • Define and understandUNIX operating system and basic commands
  • Write a complete program in Bash Shell Programming.
  • Describeand understand the use of arrays.
  • Have the ability to write elementary scripts to automate tasks.
