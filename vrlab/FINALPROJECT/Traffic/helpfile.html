Click the blue truck to have it circle around a predefined path.

There are different camera viewpoints. Cameras 1-3 are overviews of the city and the rest are user specific cameras. You will have to manually select the camera corresponding to the user you've selected.

The HUD
  • Turn Speed - How fast it is to turn around
  • Orientation - Which direction you are facing in degrees
  • Speed - How fast you can go
You will also see instructions how to get this help file up along with the user selected and any errors (if any).
