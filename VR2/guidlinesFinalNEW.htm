COSC 829: Advanced Virtual Reality Systems

Project Deliverables and Evaluation


Evaluation of group projects for the final grade will be based on the following items:


Project Report:
Each group will create a project report in detail and contains as a minimum text, images, and the VRML model of the virtual environment developed. The content of the written part should contain the items from the project proposal, but will most probably differ in several points and will be more elaborate. In addition, the report should describe the problems encountered, the final shortcomings of the applications, and recommendations for further improvements.


Project Presentation:
Each group will give a 15 to 20 minute project presentation in front of the class.


Project Demonstration and Mid Project Demos:
Each student will give a project demonstration to the teaching staff. All students must attend.

Project Deliverables

  • USE VIZARD or UNITY 3D
  • The project must include:
      1. Vision: Use of textures and 3D models to provide detailed information in the project.
      2. Sound: Use speech, music or ambient sounds to help provide information about this place.
      3. Animation: Use at least three animated object in the project.
      4. Interactivity: Use at least five user-triggered events in the environment.
      5. Characters: Have animated agents (path following behavior) in the environment.
      6. Sensors: Use at least three different types of sensors ( Proximity, Time, Touch, and Visibility) in the project.
      7. Player: Add a Player Controller to the Scene: First Person Controller or 3rd Person Controller
      8. Multi-user Environment: More than two people to be present in the same environment.
      9. CUSTOM avatar: (car/fish, etc.) and atleast 1 INBUILT avatar in vizard/Unity3D to create the multi-agent system.
      • Extra Credit: Oculus integration or 5DT data glove integration.
      • Extra Credit: Implement different behviors (selfish, altruistic, learning, adaptive, etc.) in the environment through a user menu (number of agents assignment)
