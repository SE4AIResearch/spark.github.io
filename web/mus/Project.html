RESEARCH OBJECTIVE
It  displays a Digital Diary that can do  the functions of a Calculator, Clock, Alarm Clock, Address Book, and has online Help.
It  Performs the Basic functions of a Calculator such as Addition,  Subtraction, Multiplication, Division. The numbers are entered by clicking the buttons on the Number pad in the Applet using the Mouse.
It has Clock which Displays the current System's time when the clock Button in the applet is clicked. Press Clear to close clock Alarm Clock. Using the alarm clock one can set the time at which the alarm needs to ring. Click on the Set Clock button and enter the time at which you want to set the alarm.
It has Alarm clock which can be set to specific time. It sounds an alarm when the system time becomes equal to the set time.The numbers are entered by clicking the buttons on the Number pad in the Applet using the Mouse.
It has Address Book which is Used for storing the information such as name address, telephone number of people. Click on the Address Book and enter the data and press store. To get information stored press the retreive button
