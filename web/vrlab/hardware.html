Hardware

The University's Virtual Reality Laboratory employs powerful computing hardware for its projects.

  • Apple G5-based cluster computer, X-Seed, that supports the department’s educational and research activities
  • 3D Scanner
  • Workstations with Intel (R) Core (TM)2 Quad CPU, Q6600@2.40GHz, 7.93GB RAM, 465 GB hard disk.

Software

  • Visual C/C++
  • OpenGL
  • 3D StudioMax
  • FormZ
  • VRML pad 2.1
  • CosmoPlayer

To view our three-dimensional VRML 2.0 models, we recommend the CosmoPlayer plug-in for Internet Explorer, Firefox, Mozilla, and other Web browsers. The CosmoPlayer can be downloaded from this NIST Web site, from karmanaut, or from other places. Please refer to these latest Download and Install Instructions from NIST.
