VRML REFLECTION MODEL


  • Reflection properties are defined in Material node



  • VRML Material Node


    • diffuseColor
      an object's base color,
      default (0.8, 0.8, 0.8)
      - corresponds to diffuse reflection coefficient kd

    ambientIntensity
      (a single value)
      controls reflected color of an object in response to the ambient light in the scene,
      default (0.2), range from 0.0 to 1.0
      - reflected ambient color = diffuseColor * ambientIntensity
      - this product corresponds to ambient reflection coefficient ka

    specularColor
      reflected color of an object's highlights,
      default (0.0, 0.0, 0.0)
      - corresponds to specular reflection coefficient ks

    shininess
      (a single value)
      degree of shininess of an object's surface, ranging from 0.0 for a diffuse surface with no shininess to 1.0 for a highly polished surface,
      default (0.2)
      - corresponds to specular reflection exponent n (normalized to range 0-1)

    emissiveColor
      light produced by an object, makes object 'glow',
      default (0.0, 0.0, 0.0)
      - corresponds to emissive intensity Ie

    transparency
      (a single value)
      degree of transparency of an object's surface, ranging from 0.0 for an opaque surface to 1.0 for a completely transparent surface,
      default (0.0)



  • Object Appearance


    • Final appearance of on object depends on
    - shape of object (surface normals)
    - definition of light sources and ambient light in scene
    - viewing direction
    - combination of all materials properties
    - shading algorithm used (see next chapter)
