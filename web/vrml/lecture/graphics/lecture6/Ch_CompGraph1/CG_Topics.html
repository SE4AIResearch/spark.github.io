COMPUTER GRAPHICS TOPICS

Computer Graphics (CG) is a field that covers many topics. Most of these topics are important to the development of virtual environments. The following lists the general subject areas of CG.

  • Graphics Hardware, Input & Output Devices


  • Raster Graphics Algorithms

    • scan conversion, antialiasing, filling, clipping, ...

  • Graphics Programming Methods


  • Interactivity, Graphics User Interfaces (GUI)


  • Geometrical Transformations


  • 3D Viewing

    • projections, perspective transformation, hidden surface removal, ...

  • Illumination and Shading


  • Geometric Modeling


  • Animation


  • Image Processing


  • Data Structures


  • Standardization


Major Text Book (the CG 'bible') by Foley, van Dam, Feiner, Hughes:
Computer Graphics, Principles and Practice, Addison Wesley
