RASTER SCAN DEVICE


  • Operates always in refresh mode

    • - refresh rate between 60 Hz and 120 Hz (frames per second)
    - image on monitor is continuously redrawn from the current content of the refresh buffer

  • A refresh buffer stores the current bitmap

    • - also called video RAM (Random Access Memory)
    - the content of the bitmap is created/modified by the raster algorithm of the graphics application program

  • For black&white display: one bit per pixel

    • - bitmap consists of a single bitplane
    - the 'rectangular' bitmap corresponds to the rectangular display surface on the monitor
    - each bit in the bitmap corresponds to a single pixel (dot) on the screen

  • For color display:
    multiple bits per pixel organized in bitplanes

    • (see graph on the following page)

      4 bitplanes = 4 bits per pixel = 16 different colors
      8 bitplanes = 8 bits per pixel = 256 different colors
      24 bitplanes = 16.8 million different colors
      in general: k bitplanes = 2^k different colors





  • Resolution:


    • - number of pixels in x and y direction (M x N)
    - determines the quality of the graphics representation
    - Macintosh Plus: 512 x 342 pixels (1 bitplane)
    - NTSC Video Format: 640 x 480 pixels
    - Typical Silicon Graphics: 1280 x 1024 pixels (24 bitplanes)
    - HDTV Format: 1920 x 1200 pixels
    - absolute resolution: measured in pixels per inch

      Example: 1280 x 1024 resolution and 16.8 million colors or 24 bitplanes
      required RAM for refresh buffer:
      1280 x 1024 x 24 = 31,457,280 bits
      or 31,457,280 / 8 = 3,932,160 bytes
      or 3,932,160 / 1024 = 3,840 KB (kilobytes)
      or 3,840 / 1024 = 3.75 MB (megabytes)


  • Graphics Output Primitives:


    • - Characters (display attributes: font, size, color, ...)

    - Lines (display attributes: line width, style, color, ...)
      Curves are approximated by Polylines.
      A Polyline is a sequence of piecewise linear segments.

    - Fill Area (display attributes: color, pattern, texture, ...)
      Area boundaries are, in general, Polygons (with linear edges).
