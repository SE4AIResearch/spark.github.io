COSC 590/490: Selected Topics in COSC: Virtual Reality and its Applications

Project Deliverables and Evaluation


Evaluation of group projects for the final grade will be based on the following items:


Project Report:
Each group will create a project report in detail and contains as a minimum text, images, and the VRML model of the virtual environment developed. The content of the written part should contain the items from the project proposal, but will most probably differ in several points and will be more elaborate. In addition, the report should describe the problems encountered, the final shortcomings of the applications, and recommendations for further improvements.


Project Presentation:
Each group will give a 15 to 20 minute project presentation in front of the class.


Project Demonstration and Mid Project Demos:
Each group will give a project demonstration to the teaching staff. Students from other groups must attend.


Individual Contributions and Cross-Evaluation:
Each student of a project group will submit a written statement with the following information:

  • the student's individual contribution to the project
  • an evaluation of yourself and the other group members

Forms for these statements will be provided.
