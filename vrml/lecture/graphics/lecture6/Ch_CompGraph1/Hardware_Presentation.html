HARDWARE REPRESENTATION


  • Hardware Presentation of Color


    • - RGB model with 0-1 intensities is device independent

    - intensities must be finally expressed as integers

    - example:

      8 bits (1 byte) per intensity allows for 256 values
      24 bits (3 bytes) are needed for the three intensities
      24 bitplanes are required, 8 for each intensity
      result: a total of 256 x 256 x 256 = 16.8 million colors can be expressed



    - a lower number of bitplanes requires a Color Table (or Video Look-Up Table)
    - the pixel presentation from the available bitplanes is interpreted as a pointer
    - the pointer points into the Color Table where the color is defined in 24 bits
