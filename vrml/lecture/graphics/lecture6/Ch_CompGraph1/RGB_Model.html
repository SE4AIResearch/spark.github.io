THE RGB COLOR MODEL


  • RGB (red, green, blue) Color Model

    • - a simplified, hardware-oriented color model
    - most commonly used in computer graphics
    - other models:
      CMY (cyan, magenta, yellow) used by color printers
      HSV (hue, saturation, value)
      HLS (hue, lightness, saturation)


  • A Color is Defined by an (R,G,B) Triple

    • - primary colors: red, green, blue
    - intensity for each component: between 0.0 and 1.0
    - limits: zero intensity (0.0) to full intensity (1.0)
    - intensities are added together to form the resulting color


  • The RGB Color Cube

    • - Cartesian coordinate system with R, G, and B axes
    - unit cube defined by limits (0 to 1) for each axis
    - all colors lie within the cube or on the faces of the cube






  • Primary Colors:

    • (on main axis)
    - full red : ( 1, 0, 0 )
    - full green : ( 0, 1, 0 )
    - full blue : ( 0, 0, 1 )
  • Complementary Colors:

    • (opposite to primary colors)
    - cyan : ( 0, 1, 1 )
    - magenta : ( 1, 0, 1 )
    - yellow : ( 1, 1, 0 )
  • Black and White:

    • (at origin and opposite)
    - black : ( 0, 0, 0 )
    - white : ( 1, 1, 1 )
  • Gray Shades:

    • - all gray shades: on diagonal from black to white
    - medium gray: at center of cube at (0.5, 0.5, 0.5)
