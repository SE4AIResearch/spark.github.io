VIEWING PIPELINE


  • The 3D Viewing Pipeline


    • A multi-step process is used to convert the view of a 3D model into a 2D pixel representation in the bitmap from where it is displayed on a physical output device.

    Although called a 'pipeline' process (suggesting sequential steps), step execution is in part sequential, in part parallel, and differs widely among viewing software packages and hardware configurations.

    High-end graphics computers support this process through special hardware architecture (geometry engine, graphics pipe).


  • Basic Steps


    • Transformation

    • Transform objects from world coordinates to viewing coordinates
      (the viewing coordinate system is connected with view point, view plane, viewing frustum, etc., and moves with the user)

    Clipping

    • Eliminate objects that are completely outside viewing frustum (view culling)


    • Eliminate back-faces (back-face culling)


    • Transform viewing frustum into a rectangular parallelepiped (view volume)
      (see graph at below)


    • Clip all objects against boundaries of view volume


    Projection

    • Project remaining/clipped objects onto the view plane (projection plane)

    Bitmap Creation and Display

    • Map view plane into physical device coordinates (resolution of bitmap)


    • Apply scan conversion and fill polygon algorithms


    • Apply illumination and shading techniques (see next section)


    • Use z-buffer for hidden surface removal


    • Apply antialiasing techniques


    • Display bitmap on output device
