Imaging Room

 


· Use of textures:

Imaging room is modeled as a labyrinth to make it interesting and fun so the walls are textures with images ranging from year 1900- 2000 and it has an interactive quiz session in which the viewer is been asked questions on what he noticed in the imaging room .It has sensors attached to the textures which are scripted so that whenever the textures are touched they are changed to the textures belonging to the next 25 years .

Historical Imaging
Imaging room displays array of images for year from 1900 -1925. After that there is a quiz question and after answering the quiz, the images are changed to the new set of images for next 25 years. This process conitnues till the year 2000.

Recomendations for further improvements

Future applications of this project will include the development of a this project as a full time game with point systems where points can be gained inside the atrium and in individual museum rooms. The user will be allowed to enter the game room only when he has gained enough points from previous museum rooms. The imaging room will incorporate informative exhibition about historical and technical background with game quiz concept.
