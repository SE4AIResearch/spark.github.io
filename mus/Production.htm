PRODUCTION ROOM

 


Shows the production line in manufacturing. The machine displays the gear fixed on the rotating shaft and lathe machine-drilling hole in it. The lathe machine has sensors through which the animation of the lathe machine takes place. There is a start sensor to start the lath machine and there is stop sensor to stop the machine. Also there is animation of the robot performing welding for the car in the production line.

 

Software and Hardware Used
